## Getting Started
### Run the full server:
1. Install the module with: `npm install js-practice`
1. Then run: `node server.js`

### Use the library in your own server
```javascript
var PayServer = require('js-practice');
var server = new PayServer('.',3000);
js_practice.create_listender();
```
